🛠️ 部署工具

本章介绍各种一键脚本、可视化面板和管理工具，帮助你快速在 VPS 上部署代理服务。

工具分类

一键脚本 vs 可视化面板

类型 优点 缺点 适合人群
一键脚本
  • ✅ 安装快速
  • ✅ 资源占用少
  • ✅ 纯净安装
  • ⚠️ 命令行操作
  • ⚠️ 管理不直观

  • ✅ 图形化界面
  • ✅ 操作简单

  • ⚠️ 资源占用多
  • ⚠️ 额外风险
 新手、需要管理多用户

推荐一键脚本

1. Xray-install（官方脚本）推荐

特点

  • ✅ Xray 官方维护
  • ✅ 纯净安装
  • ✅ 支持 VLESS、VMess、Trojan
  • ✅ 支持 Reality 协议

安装命令

bash -c "$(curl -L https://github.com/XTLS/Xray-install/raw/main/install-release.sh)" @ install

项目地址

GitHub: XTLS/Xray-install

2. v2ray-agent（多协议）推荐

特点

  • ✅ 支持多协议共存（Xray、V2Ray、Trojan、NaiveProxy）
  • ✅ 中文交互界面
  • ✅ 支持 IPv6
  • ✅ 支持 CDN
  • ✅ 功能最全面

安装命令

wget -P /root -N --no-check-certificate "https://raw.githubusercontent.com/mack-a/v2ray-agent/master/install.sh" && chmod 700 /root/install.sh && /root/install.sh

项目地址

GitHub: mack-a/v2ray-agent

3. Hysteria2-install（极速协议）

特点

  • ⚡ 专注 Hysteria2 协议
  • ✅ 自动配置 BBR 加速
  • ✅ 自动申请 TLS 证书
  • ✅ 配置简单

安装命令

bash <(curl -fsSL https://get.hy2.sh/)

项目地址

GitHub: apernet/hysteria

可视化管理面板

1. 3X-UI（最流行）强烈推荐

核心特性

支持协议 VMess、VLESS、Trojan、Shadowsocks、Reality
核心引擎 Xray-core
界面 中文 Web 面板
功能
  • ✅ 多用户管理
  • ✅ 流量统计
  • ✅ 订阅链接生成
  • ✅ Telegram Bot 通知
  • ✅ 在线终端

安装命令

bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)

默认访问

安装完成后访问：http://your-vps-ip:54321

默认用户名：admin | 默认密码：admin

项目地址

GitHub: MHSanaei/3x-ui

2. S-UI（Sing-box 面板）

核心特性

支持协议 VLESS、Reality、Trojan、Shadowsocks、Hysteria2、TUIC
核心引擎 Sing-box（更现代）
特点
  • ✅ 支持最新协议
  • ✅ 性能优秀
  • ✅ 配置灵活
  • ⚠️ 相对较新

安装命令

bash <(curl -Ls https://raw.githubusercontent.com/alireza0/s-ui/master/install.sh)

项目地址

GitHub: alireza0/s-ui

部署流程示例（3X-UI）

步骤 1: 连接 VPS

# 使用 SSH 连接
ssh root@your-vps-ip

# 输入密码登录

步骤 2: 安装面板

bash <(curl -Ls https://raw.githubusercontent.com/mhsanaei/3x-ui/master/install.sh)

步骤 3: 访问面板

打开浏览器访问：http://your-vps-ip:54321

使用默认账号密码登录：admin / admin

步骤 4: 创建入站

  1. 点击"入站列表" → "添加入站"
  2. 选择协议（推荐 VLESS + Reality）
  3. 配置端口和参数
  4. 点击"添加"保存

步骤 5: 获取连接信息

点击"操作" → "二维码"或"链接"，导入到客户端使用

安全提示

⚠️ 使用面板的注意事项

  • 🔒 立即修改默认密码 - 安装后第一件事
  • 🔐 修改默认端口 - 避免使用 54321 等常见端口
  • 🛡️ 启用防火墙 - 只开放必要端口
  • 🚫 不要使用弱密码 - 使用强密码或密钥
  • 💾 定期备份配置 - 防止数据丢失
  • 🔄 及时更新 - 保持软件最新版本

常见问题

Q: 脚本安装失败怎么办？

A: 常见原因和解决方案：

  • 检查网络连接是否正常
  • 确保系统是 Ubuntu/Debian（推荐 Ubuntu 22.04）
  • 检查是否已安装其他代理软件（可能冲突）
  • 查看安装日志，寻找具体错误信息

Q: 面板无法访问？

A: 检查以下几点：

  • 确认防火墙已开放面板端口
  • 检查服务是否正常运行：systemctl status x-ui
  • 确认 IP 地址和端口正确
  • 尝试使用 http 而非 https
