❓ 常见问题与故障排查

新手常见问题

Q1: 第一次搭建，推荐什么方案？

A: 推荐方案：

  1. VPS: Vultr 日本/新加坡节点（$6/月）
  2. 系统: Ubuntu 22.04 LTS
  3. 工具: 3X-UI 面板
  4. 协议: VLESS + Reality（无需域名）
  5. 客户端: v2rayN (Windows) / Shadowrocket (iOS)

Q2: 需要懂编程吗？

A: 不需要。使用一键脚本和图形化面板，只需要：

  • 会用 SSH 连接服务器
  • 会复制粘贴命令
  • 基本的操作系统使用能力

Q3: 需要域名吗？

A: 看协议：

  • ✅ 不需要域名: VLESS Reality、Shadowsocks、Hysteria2
  • ❌ 需要域名: Trojan、VMess + WebSocket + TLS

新手推荐 Reality，无需域名，隐蔽性最强。

Q4: 自建 vs 机场，选哪个？

对比项 自建 机场
隐私 ✅ 完全私有 ❌ 流量被监控
稳定性 ✅ 自己控制 ⚠️ 可能跑路
成本 $5-10/月 $5-20/月
技术门槛 ⚠️ 需要基本技能 ✅ 开箱即用
节点数量 自己搭建几个 ✅ 通常很多节点

建议: 注重隐私和技术学习 → 自建；只想快速使用 → 机场。

连接问题

Q5: 无法连接服务器

A: 排查步骤：

  1. 测试 IP 是否被墙 
    ping your-vps-ip
  2. 检查端口是否开放 
    telnet your-vps-ip 443
  3. 检查服务是否运行 
    systemctl status xray
systemctl status x-ui
  4. 检查防火墙 
    ufw status
ufw allow 443/tcp
  5. 检查配置是否正确
    • IP 地址、端口、UUID、协议是否匹配

Q6: 连接后网速很慢

A: 可能原因和解决：

原因 解决方案
线路拥堵（晚高峰） 更换为 CN2 GIA 或 IPLC 线路
VPS 性能不足 升级配置或更换商家
QoS 限速 使用伪装协议（Reality、Trojan）
未启用 BBR 安装 BBR 加速
服务器位置远 选择香港、日本等近距离节点

Q7: 频繁断线

A: 检查：

  • 协议是否被识别（改用 Reality、Trojan）
  • IP 是否被轻度封锁（测试端口连通性）
  • 客户端是否开启了 "自动重连"
  • VPS 是否稳定（查看 uptime 和 CPU 使用率）

配置问题

Q8: 忘记面板密码怎么办？

A: 3X-UI 重置方法：

# 重置管理员账号
x-ui

# 选择 6 - 修改账号密码
# 输入新的用户名和密码

Q9: 如何备份配置？

# 备份重要文件
tar -czf backup.tar.gz /etc/xray /etc/x-ui /etc/nginx

# 下载到本地
scp root@your-vps-ip:backup.tar.gz ./

Q10: 如何迁移到新服务器？

  1. 在旧服务器备份配置
  2. 在新服务器安装相同版本的面板
  3. 恢复配置文件
  4. 修改客户端的服务器 IP
  5. 测试连接后删除旧服务器

安全问题

Q11: SSH 无法登录

A: 可能原因：

  • 修改端口后忘记（查看商家控制台的 Console 登录）
  • 禁用密码登录但密钥配置错误
  • 防火墙阻止了 SSH 端口
  • fail2ban 误封了你的 IP

解决: 使用商家提供的 VNC/Console 登录修复。

Q12: 服务器被入侵怎么办？

  1. 立即断开服务器网络
  2. 备份重要数据
  3. 重装系统
  4. 加固安全配置（修改 SSH 端口、密钥登录、防火墙）
  5. 分析入侵原因

协议与客户端

Q13: Reality vs Trojan，选哪个？

对比项 Reality Trojan
需要域名 ✅ 不需要 ❌ 需要
隐蔽性 ⭐⭐⭐⭐⭐ ⭐⭐⭐⭐
配置难度 中等 中等
客户端支持 需要新版本 支持广泛

建议: 没有域名 → Reality；有域名 → 两者都可以，Reality 更强。

Q14: iOS 客户端哪个最好？

A: 推荐顺序：

  1. Shadowrocket ($2.99) - 最佳选择，功能全面
  2. Quantumult X ($7.99) - 功能最强，适合进阶
  3. Loon ($5.99) - 界面美观，配置灵活

新手直接买 Shadowrocket 即可。

Q15: 如何在路由器上配置代理？

A: 方案：

  • 方案 1: 刷梅林固件 + 安装 Clash
  • 方案 2: 使用软路由（OpenWrt + PassWall）
  • 方案 3: 树莓派 + 透明代理

具体教程需要另外学习，建议先掌握基础配置。

费用与选择

Q16: 每月预算多少合适？

预算 配置 适用人群
$5-10/月 基础 VPS + 普通线路 轻度使用、学习测试
$10-20/月 CN2 GT 或高配 VPS 日常使用、看视频
$20-50/月 CN2 GIA 或多节点 重度使用、追求体验
$50+/月 IPLC 专线 企业、极致稳定

Q17: 流量够用吗？

A: 流量消耗参考：

  • 浏览网页：10-50 MB/小时
  • 1080P 视频：3-5 GB/小时
  • 4K 视频：15-20 GB/小时
  • 下载文件：按文件大小

个人使用 500GB/月 通常够用，重度看视频需要 1-2TB。

故障排查流程

标准排查步骤

  1. 确认基础连通性
    • ping 服务器 IP
    • telnet 服务器端口
  2. 检查服务状态 
    systemctl status xray
journalctl -u xray -n 50
  3. 验证配置正确
    • 对比服务端和客户端配置
    • 检查 UUID、端口、协议
  4. 查看日志 
    tail -f /var/log/xray/access.log
tail -f /var/log/xray/error.log
  5. 测试替代方案
    • 换个客户端试试
    • 换个节点试试
